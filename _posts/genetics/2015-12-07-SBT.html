--- layout: post title: SBT categories : [Genetics, _Scala] comments: True --- Het programma zal gemaakt worden in Java en (zoveel mogelijk) Scala. Beide talen draaien op dezelfde JVM Java Virtual Machine en kunnen door elkaar gebruikt worden.

Als IDE Integrated Developement Environment wordt Eclipse gebruikt, maar daarin kunt u uw eigen voorkeur volgen.

Jammer genoeg zijn Applets Java of Scala programma's die in een Browser-window draaien niet meer praktisch te gebruiken nadat Oracle ze om security-redenen de nek heeft omgedraaid. Doodzonde!
U zult dus zelf de code moeten installeren en draaien als applicatie. Het programma zal op GitHub worden geplaatst.

Eenvoudige Scala programma's kunnen na installatie van de Scala IDE voor Eclipse worden opgezet zonder verdere kennis van SBT. Scala Build Tool
Ik ben echter van plan om het programma op te bouwen uit meerdere projecten, en daarvoor zal ik me eerst moeten verdiepen in SBT: de eerste technische hobbel.
